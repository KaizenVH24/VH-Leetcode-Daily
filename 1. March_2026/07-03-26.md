### 1888. Minimum Number of Flips to Make the Binary String Alternating

# Intuition

An alternating binary string can only follow two patterns:  
`010101...` or `101010...`.

The tricky part in this problem is that we are allowed to rotate the string by moving the first character to the end. Instead of physically rotating the string multiple times, we can observe that every rotation of `s` will appear as a substring of length `n` inside `s + s`.

So if we double the string, we can check all possible rotations using a sliding window of size `n`.

For each window, we simply count how many characters differ from the two valid alternating patterns and take the minimum number of flips needed.

---

# Approach

1. Let `n` be the length of the string and create `s + s` so that every rotation appears as a substring.
2. Build two alternating reference patterns:
   - One starting with `'0'`
   - One starting with `'1'`
3. Use a sliding window of length `n` across the doubled string.
4. While expanding the window:
   - Track mismatches with both alternating patterns.
5. If the window size becomes larger than `n`, move the left pointer and update mismatch counts.
6. Whenever the window size equals `n`, update the answer with the minimum mismatch count.

This effectively checks all rotations without actually rotating the string.

---

# Complexity

- **Time complexity:**  
  $$O(n)$$  
  We traverse the doubled string once using a sliding window.

- **Space complexity:**  
  $$O(n)$$  
  Extra space is used for the two alternating pattern arrays.

---

# Code
```python
class Solution:
    def minFlips(self, s: str) -> int:

        n = len(s)
        s = s + s
        
        alt1 = []
        alt2 = []
        
        for i in range(len(s)):
            alt1.append('0' if i % 2 == 0 else '1')
            alt2.append('1' if i % 2 == 0 else '0')
        
        res = float('inf')
        diff1 = diff2 = 0
        left = 0
        
        for right in range(len(s)):
            if s[right] != alt1[right]:
                diff1 += 1
            if s[right] != alt2[right]:
                diff2 += 1
            
            if right - left + 1 > n:
                if s[left] != alt1[left]:
                    diff1 -= 1
                if s[left] != alt2[left]:
                    diff2 -= 1
                left += 1
            
            if right - left + 1 == n:
                res = min(res, diff1, diff2)
        
        return res
```
