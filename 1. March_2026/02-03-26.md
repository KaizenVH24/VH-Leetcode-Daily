## 1536. Minimum Swaps to Arrange a Binary Grid

--- 
## Code
``` python
class Solution:
    def minSwaps(self, grid):
        n = len(grid)
        trailing = []

        for row in grid:
            count = 0
            for val in reversed(row):
                if val == 0:
                    count += 1
                else:
                    break
            trailing.append(count)

        swaps = 0

        for i in range(n):
            needed = n - 1 - i
            j = i

            while j < n and trailing[j] < needed:
                j += 1

            if j == n:
                return -1

            while j > i:
                trailing[j], trailing[j - 1] = trailing[j - 1], trailing[j]
                swaps += 1
                j -= 1

        return swaps
```
---

## Time & Space Complexity: O(n²) and O(n) respectively
