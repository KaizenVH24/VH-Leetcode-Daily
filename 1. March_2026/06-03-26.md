### 1784. Check if Binary String Has at Most One Segment of Ones


# Intuition

Since the string represents a binary number without leading zeros, it always starts with `'1'`.  
For the condition to hold, all the `1`s must appear in a single continuous block at the beginning.  

Once we encounter a `0`, we should never see another `1` again.  
If a `1` appears after a `0`, that means there is another segment of `1`s, which violates the rule.

So the only pattern that would break the condition is when the string goes from `1 → 0 → 1`.

---

# Approach

The transition `1 → 0 → 1` would show up in the string as the substring `"01"` followed later by another `1`.  
But if we look closely, the moment `"01"` appears, we already know that a block of `1`s ended and a `0` started.

If another `1` appears later, the string will contain `"01"` again.

So checking whether `"01"` exists in the string is enough to detect a second segment of ones.  
If `"01"` is not present, all the `1`s are grouped together.

---

# Complexity

- **Time complexity:**  
  $$O(n)$$  
  Checking whether a substring exists requires scanning the string once.

- **Space complexity:**  
  $$O(1)$$  
  No extra memory is required apart from a few variables.

---

# Code
```python
class Solution:
    def checkOnesSegment(self, s: str) -> bool:
        return "01" not in s
```
