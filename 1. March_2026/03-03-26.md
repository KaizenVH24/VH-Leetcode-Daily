### Recursive Approach

```python
lass Solution:
    def findKthBit(self, n: int, k: int) -> str:
        if n == 1:
            return "0"
        
        length = (1 << n) - 1
        mid = (length // 2) + 1
        
        if k == mid:
            return "1"
        elif k < mid:
            return self.findKthBit(n - 1, k)
        else:
            mirrored = self.findKthBit(n - 1, length - k + 1)
            return "1" if mirrored == "0" else "0"
```
## Complexity
Time: O(n)
Space: O(n) (recursion stack)

### Iterative Approach ( optimized )

```python
class Solution:
    def findKthBit(self, n: int, k: int) -> str:
        invert = False
        
        while n > 1:
            length = (1 << n) - 1
            mid = (length // 2) + 1
            
            if k == mid:
                return "0" if invert else "1"
            
            if k > mid:
                k = length - k + 1
                invert = not invert
            
            n -= 1
        
        return "1" if invert else "0"
```

## Complexity

Time: O(n)
Space: O(1)
