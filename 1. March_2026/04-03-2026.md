# Intuition

A position `(i, j)` can only be special if the value at that position is `1`.  
But that alone isn’t enough — every other element in the same row and column must be `0`.

So if a row contains more than one `1`, none of those positions can be special.  
The same rule applies to columns.

Because of this, it makes sense to first know how many `1`s appear in each row and each column before checking individual positions.

---

# Approach

The idea is to break the problem into two passes over the matrix.

First, count how many `1`s appear in every row and every column.  
This gives us two helper arrays: one storing row counts and the other storing column counts.

Next, go through the matrix again.  
Whenever a cell contains `1`, check:

- The row it belongs to has exactly one `1`
- The column it belongs to has exactly one `1`

If both conditions hold, that position is special and we increase the counter.

---

# Complexity

- **Time complexity:**  
  $$O(m \times n)$$  
  The matrix is scanned twice.

- **Space complexity:**  
  $$O(m + n)$$  
  Extra arrays are used to store row and column counts.

---

# Code
```python
class Solution:
    def numSpecial(self, mat: List[List[int]]) -> int:

        m, n = len(mat), len(mat[0])
        row_count = [0] * m
        col_count = [0] * n
        
        for i in range(m):
            for j in range(n):
                if mat[i][j] == 1:
                    row_count[i] += 1
                    col_count[j] += 1
        
        special = 0
        
        for i in range(m):
            for j in range(n):
                if mat[i][j] == 1 and row_count[i] == 1 and col_count[j] == 1:
                    special += 1
        
        return special
```
